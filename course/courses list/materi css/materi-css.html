CSS Pengenalan


Apa itu CSS?

CSS atau Cascade Style Sheet merupakan sebuah bahasa yang mengatur tampilan dari web sehingga menjadi lebih menarik dan indah. Ini merupakan bahasa kedua yang harus dipelajari untuk menjadi seorang Web Developer. Dengan CSS kita dapat mengatur tata letak/layout, mengatur ukuran font, warna, garis, dan lain-lain.

Kok pake CSS?

Tampilan dari website tanpa CSS dan dengan CSS berbeda jauh, baik dari segi tata letak/layout, warna, atau desain.

Kok pake CSS? Bukannya di HTML bisa ngatur warna, font, dan lainnya?
HTML tidak pernah dimaksudkan untuk mengatur bentuk dan ukuran website. HTML dibuat untuk menggambarkan kerangka atau konten dari sebuah halaman web. Ketika tag seperti font muncul, dan atribut warna ditambahkan pada spesifikasi HTML 3.2, itu memulai mimpi buruk pada web developers. Kode HTML dipenuhi dengan pengaturan font dan warna disetiap baris, sehingga proses menjadi panjang.
Untuk menyelesaikan permasalahan ini maka World Wide Web Consortium (W3C) membuat CSS. Dengan memakai CSS tidak perlu mengatur font dan warna disetiap baris.

Jika belum mengetahui HTML, disarankan untuk membaca materi HTML

Selektor

Selektor adalah kata kunci untuk memilih elemen HTML yang ingin diubah style nya. Selektor secara sederhana memilih elemen berdasarkan nama, id, class.


Selektor CSS

Selektor nama

Selektor berdasarkan nama, berarti kita memanggil elemen HTML dengan nama tag atau elemennya.
Contoh penggunaan selektor berdasarkan nama elemen HTML

Jadi, itu berarti kita memilih semua elemen <h1>, lalu warna teks dirubah menjadi white (putih), dan teks <h1> berada di tengah. Selektor berdasarkan nama elemen mencakup semua tag atau elemen yang ada dalam HTML.


Selektor id

Id digunakan untuk memberi tanda kepada sebuah elemen yang sifatnya unik atau tidak dapat dimiliki elemen lain. Id sama halnya dengan KTP yang hanya dapat dimiliki oleh satu orang saja. Untuk mengakses atau memanggil id dalam CSS menggunakan tanda hash atau tagar (#).

Contoh penggunaan selektor berdasarkan id

Jadi, itu berarti kita memilih elemen yang memiliki id judul dirubah warna menjadi putih, dan warna latar menjadi hitam.


Selektor class

Class adalah selektor yang digunakan untuk elemen yang lebih dari satu. Biasanya class digunakan untuk menggabungkan elemen yang mempunyai sifat yang sama, dan class juga digunakan untuk mempermudah mengatur desain dalam CSS karena bisa mengatur lebih dari satu elemen hanya dengan mengubah satu class. Untuk mengakses atau memanggil class dalam CSS menggunakan tanda titik (.).

Contoh penggunaan selektor berdasarkan class

Jadi, itu berarti kita memilih semua elemen yang memiliki class "container" dirubah menjadi warna putih, dan warna latar menjadi hitam.
