HTML Pengenalan


Apa itu HTML?

Sesuai namanya, Hyper Text Markup Language atau sering disingkat HTML merupakan sebuah bahasa markup yang memberi teks kepada sebuah halaman web. HTML mengatur isi dari halaman sebuah website. Bisa dibilang, HTML merupakan fondasi dasar dari sebuah halaman web

HyperText mengacu pada tautan yang menghubungkan sebuah halaman ke halaman lainnya. Baik itu pada sebuah website, maupun antara website

HTML menggunakan Markup sebagai anotasi dari elemen-elemen website seperti header, h1, p, section, footer, dan lainnya

HTML

Tag HTML merupakan tag pembuka pada setiap file html. Tag ini berfungsi untuk membuka dan menutup file html yang kita buat

Head

Merupakan sebuah tag tempat kita menyimpan beberapa komponen yang tidak akan ditampilkan ke halaman website kita, seperti title, link menuju css, tag meta, dan lainnya

Body

Merupakan tempat kita menulis code yang akan ditunjukkan pada halaman HTML, mulai dari teks, gambar, audio, dan lainnya

Atribut HTML

Pada setiap tag-tag html, terpasang sebuah atribut. Baik itu secara default atau kita yang mengisikannya sendiri secara manual. Ada beberapa atribut yang ada pada tag-tag html, seperti class, id, href, src, dan masih banyak lagi.
